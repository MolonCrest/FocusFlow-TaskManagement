# FocusFlow Task Manager

## Project Title
FocusFlow Task Manager

## Project Type
Task Manager with Categories

## Project Description
FocusFlow is a browser-based task manager that I made with HTML, CSS, and JavaScript. The main idea of the app is to help keep track of different responsibilities, such as school work, personal tasks, work tasks, and other things that need to be finished. Each task can have a title, category, priority, due date, notes, and a completion status.

I tried to keep this version useful without making it way more complicated than it needed to be. The JavaScript is shorter and more direct now, but the app still has enough features to work like a real small web application instead of just being a quick demo. The form, task board, search tools, filters, sorting options, progress section, and saved data all connect together and update the page as the user interacts with it.

## Tools and Languages Used
- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

## Major Interactive Features
- Add a task with a title, category, priority, due date, and notes
- Edit an existing task instead of having to delete it and make it again
- Mark tasks as complete, or reopen them if they still need work
- Delete tasks from the task board
- Search through tasks by title or notes
- Filter tasks by status and category
- Sort tasks by newest, due date, or priority
- Clear all completed tasks when the list starts getting too full
- View dashboard totals and a progress bar that updates automatically

## Required Concepts Covered
- DOM manipulation with JavaScript
- User input handling through a form, buttons, search box, and select menus
- Arrays and objects for storing and managing the task data
- Functions organized by purpose, so the script is not just one long block of code
- Simplified JavaScript kept under 250 lines
- Client-side validation for blank titles, short titles, missing due dates, and duplicate tasks
- Conditional rendering for empty search results, completed tasks, due soon tasks, and overdue tasks
- CSS styling with a responsive layout
- localStorage as the advanced course concept, which lets the task list stay saved in the browser

## Optional Features Intentionally Left Out
This version does not include an API connection, timer system, drag and drop sorting, login system, or custom account data.
